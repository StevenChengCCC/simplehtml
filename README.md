# Course Site (Static)

This site is plain static HTML/CSS (no build step).

## Files
- `index.html` — professional landing page
- `resume.html` — résumé in HTML
- `grad.html` — graduate focus page
- `scratch/scratch.html` — scratch page
- `scratch/app.html` — mini web app (single page)

## Deploy
Host the folder as static files (AWS Amplify static hosting, GitHub Pages, etc.).
