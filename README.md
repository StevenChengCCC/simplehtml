# Simple Course Website (Static HTML)

This folder contains a minimal static website designed to satisfy the course requirements:

- Professional pages using a Bootstrap theme:
  - `index.html`
  - `resume.html` (résumé content written in HTML)
  - `grad.html` (graduate-focused extra page)
- From-scratch page (no Bootstrap):
  - `scratch/scratch.html`
  - `scratch/scratch.css`
- Small single-page web app (HTML/CSS/JS in one file):
  - `scratch/app.html`

## Deploy on AWS Amplify (recommended)

1. Push this folder to a GitHub repo (as the repo root).
2. In AWS Amplify: "New app" → "Host web app" → connect the repo.
3. Build settings:
   - Framework: **static**
   - Build command: *(none)*
   - Output directory: **/** (root)
